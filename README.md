# ✨ Happy House ( 🏆 1학기 프로젝트 우수상 )
* 사용자입장에서 원하는 아파트매물 거래 정보를 제공합니다. 
* 아파트 거래 기록 외의 정보를 추가적으로 제공합니다.


## 🎯 목표
* Web Architecture를 이해하고 활용하여 Web Project를 설계하고 구현할 수 있다.
* Spring&MyBatis Framework, Spring Boot, Vue.js를 이해하고 활용할 수 있다.
